Chapter 2 Intro to R

This is an entry-level intro to the Statistical Language R, which we wlll be using along the quarter.

2.1 A section

All chapter sections start with a second-level (##) or higher heading followed by your section title, like the sections above and below here. You can have as many as you want within a chapter.

An unnumbered section

Chapters and sections are numbered by default. To un-number a heading, add a {.unnumbered} or the shorter {-} at the end of the heading, like in this section.

The Behavioral Risk Factor Surveillance System (BRFSS) is an annual telephone survey of 350,000 people in the United States. As its name implies, the BRFSS is designed to identify risk factors in the adult population and report emerging health trends. For example, respondents are asked about their diet and weekly physical activity, their HIV/AIDS status, possible tobacco use, and even their level of healthcare coverage.

The BRFSS Web site (http://www.cdc.gov/brfss) contains a complete description of the survey, including the research questions that motivate the study and other interesting results. We will focus on a random sample of people from the BRFSS survey conducted in 2000. While there are over 200 variables in this data set, we will work with a small subset.

2.2 Loading the dataset

Begin by loading the dataset into your R workspace by running the command in the chunk below:

source("http://www.openintro.org/stat/data/cdc.R")

The file cdc.R is an R-data set, as such it has a special format, and we can use certain commands to access the data.

Note: for other types of formats, we may have to use other commands

2.3 Viewing the data set

Would it be a good idea to view this entire data set? The command View will create a table with all entries of the data set. See what happens when you run the chunk below ( the output may open in a separate tab. Make sure to come back to this tab to continue.)

View(cdc)

Ok, perhaps it’s a better idea to read parts of the data set. For example, if we want to see the variables in the set, we can run the command names. Do it in the chink below to find out what variables are stored in this data set.

names(cdc)
## [1] "genhlth"  "exerany"  "hlthplan" "smoke100" "height"   "weight"   "wtdesire"
## [8] "age"      "gender"

We can also view, for example, the first 5 rows of the data set. Run the command below to see the output

head(cdc, 5)
##     genhlth exerany hlthplan smoke100 height weight wtdesire age gender
## 1      good       0        1        0     70    175      175  77      m
## 2      good       0        1        1     64    125      115  33      f
## 3      good       1        1        1     60    105      105  49      f
## 4      good       1        1        0     66    132      124  42      f
## 5 very good       0        1        0     61    150      130  55      f

Similarly, we can also view the last 5 rows by writing the command tail(cdc, 5)
